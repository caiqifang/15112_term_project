15-112 Term Project:

Qifang Cai + ID: qcai

==========================================================================
The term project is a traditional Snake game. 
It is placed in a 3D setting, using Tkinter from Python.
The snake travels on a cube, and the users have a mini-map as a reference.
==========================================================================

To run, please have Python installed in computer and run: 
3DSnakeRun.py

If you have a Windows computer, you can click on the 3DSnakeRun.py file and run it.

*also*: make sure you have Pygame module in your Python, Pygame is used to handle the audio in this game. The graphing is done in Tkinter.

==========================================================================

To find Pygame, please Google it and download.
